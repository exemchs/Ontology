# Ontology — EXEM 온톨로지 솔루션 프로토타입

## What This Is

엑셈의 새 온톨로지 솔루션을 SK실트론에 제안하기 위한 워킹 프로토타입. 반도체 생산 공정에서 불량률을 낮추기 위해, 공정/장비/소재/불량 간 관계를 지식 그래프로 모델링하고 불량 원인을 추적할 수 있는 웹 기반 데모 애플리케이션이다.

## Core Value

불량이 발생했을 때, 지식 그래프를 탐색하여 관련 공정/장비/소재까지 원인을 시각적으로 추적할 수 있어야 한다.

### Out of Scope

- LLM/AI 기반 자동 분석 — 향후 확장 가능성 있으나 v1 프로토타입에서는 제외
- 실시간 데이터 연동 — 프로토타입은 샘플 데이터 기반
- 사용자 관리/권한 시스템 — 데모용이므로 복잡한 권한 불필요
- 모바일 앱 — 웹 데모 우선

## Context

- SK실트론: 반도체 웨이퍼 제조 기업. 생산 공정 불량률 개선이 핵심 과제.
- 엑셈은 IT 모니터링/데이터 분석 전문 기업으로, 온톨로지 솔루션은 새로운 제품 라인.
- 이 프로토타입은 제안 단계에서 솔루션의 가치를 직관적으로 보여주는 것이 목적.
- 기존 코드베이스: Next.js 16 + Supabase + Tailwind CSS 4 기반, 인증/배포 인프라 구축 완료.
- 반도체 공정 온톨로지: 웨이퍼 제조 공정(성장/절단/래핑/폴리싱/세정/검사)과 각 공정의 장비, 소재, 파라미터, 불량 유형 간 관계를 모델링.
- 데이터: 업계 표준 데이터셋 참고하여 현실적인 샘플 데이터 생성.

## Constraints

- **Tech stack**: Next.js 16 + Supabase + Tailwind CSS 4 (기존 스택 유지) — 이미 구축된 인프라 활용
- **데이터**: 샘플/더미 데이터 사용 — 실제 SK실트론 데이터 없음
- **목적**: 제안용 워킹 프로토타입 — 프로덕션 수준이 아닌 데모 수준 완성도
- **배포**: Vercel (https://ontology-eta.vercel.app) — 웹 데모로 즉시 접근 가능해야 함
